### 🔧 技术栈

`Python` `tkinter` `requests` `pandas` `GUI`

### ✨ 产品特色

- 🖱️ **开箱即用** — GUI 界面，Win/Mac 双端，无需配置编程环境
- ⚡ **接口协议采集** — 非模拟浏览器，速度快、稳定性高
- 💾 **断点续存** — 每页实时保存 CSV，不怕意外中断
- 📝 **日志完备** — 全程 log 记录，问题可回溯
- 🔑 **一机一码** — 防盗版，保障正版用户权益
